Apresentação

A Biblioteca de Pendrives de Distribuições Linux do GULECD/UFPR é uma iniciativa voltada à disseminação do conhecimento, à promoção do software livre e ao incentivo à experimentação de sistemas operacionais baseados em Linux.

O projeto consiste na disponibilização de pendrives contendo imagens de instalação de distribuições Linux, preparados de forma padronizada. Esses dispositivos podem ser compartilhados entre os participantes do grupo, permitindo acesso facilitado a diferentes distribuições sem a necessidade de download individual.

Objetivo

A biblioteca tem como objetivos principais:

  • Facilitar o acesso a distribuições Linux para estudantes e membros da comunidade;

  • Promover a cultura de software livre e colaboração;

  • Incentivar a experimentação prática em ambientes controlados

Funcionamento da Biblioteca

O funcionamento da biblioteca baseia-se no compartilhamento organizado dos pendrives entre os participantes.

  • O usuário pode retirar um pendrive na secretaria do curso de Estatística e Ciência de Dados ou diretamente com outro membro do grupo;

  • O período máximo de permanência com o pendrive é de 7 dias corridos;

  • Após esse período, o pendrive deve ser devolvido na secretaria do curso ou repassado a outro membro interessado;

  • Toda retirada e devolução deve ser registrada por meio do QR code presente no dispositivo.

Procedimento de Retirada e Devolução

Cada pendrive possui um QR code associado a um formulário de controle.

  • No momento da retirada, o usuário deve escanear o QR code e preencher o formulário correspondente;

  • No momento da devolução ou repasse, o mesmo procedimento deve ser realizado;

  • O correto preenchimento é essencial para rastreabilidade e organização da biblioteca.

Uso Adequado das Pendrives

As pendrives da biblioteca foram configuradas como dispositivos de leitura (read-only) com imagens padronizadas.

  • Não é permitido utilizar a pendrive para armazenamento de arquivos pessoais;

  • Não é permitido modificar qualquer conteúdo presente na pendrive;

  • Não é permitido utilizar a pendrive para execução de atividades que comprometam sua integridade;

  • Não é permitido utilizar a pendrive para quaisquer atividades que infrinjam a legislação brasileira;

  • A utilização deve se restringir à inicialização (boot), experimentação no ambiente Linux live e instalação das distribuições disponíveis.

Verificação de Integridade

Cada pendrive possui checksums criptográficos previamente gerados e assinados digitalmente.

  • O usuário deve, sempre que possível, verificar os checksums antes da utilização;

  • Caso qualquer divergência seja identificada, a pendrive não deve ser utilizada;

  • O problema deve ser comunicado imediatamente à coordenação do grupo;

  • Ferramentas e instruções para verificação de integridade são disponibilizadas no repositório do projeto em https://github.com/GUL-Estatistica-e-Ciencia-de-Dados-UFPR/biblioteca-distros.

Segurança e Responsabilidade do Usuário

Foram empregados os melhores esforços técnicos para garantir a integridade e segurança das pendrives, incluindo:

  • uso de imagens oficiais;

  • configuração para minimizar persistência e alterações;

  • geração e assinatura criptográfica de checksums;

  • processos de validação e verificação.

Entretanto, o usuário é responsável por verificar a integridade da pendrive antes do uso. Isso inclui:

  • validar checksums;

  • verificar assinaturas digitais dos arquivos que contém as checksums;

  • garantir que o dispositivo não foi adulterado.

Isenção de Responsabilidade

O GULECD/UFPR não se responsabiliza por:

  • perda de dados;

  • danos a hardware ou software;

  • falhas decorrentes do uso das pendrives;

  • quaisquer consequências diretas ou indiretas da utilização dos dispositivos.

O uso das pendrives é de inteira responsabilidade do usuário.

Boas Práticas Recomendadas

Para uma utilização segura e eficiente, recomenda-se:

  • realizar backup de dados importantes antes de qualquer instalação;

  • evitar uso em sistemas críticos sem validação prévia;

  • manter boas práticas de segurança digital.

Conservação dos Dispositivos

Para garantir a durabilidade das pendrives:

  • evite impactos físicos;

  • não exponha a líquidos ou temperaturas extremas;

  • não remova o dispositivo durante operações de leitura;

  • mantenha o dispositivo limpo e protegido.

Conduta e Colaboração

A biblioteca depende da colaboração dos participantes.

  • respeite os prazos de uso;

  • registre corretamente as movimentações;

  • reporte problemas técnicos;

  • contribua com sugestões de melhoria;

  • respeite os demais usuários do sistema.

Disposições Finais

O uso das pendrives implica na concordância com este regulamento.

O GULECD/UFPR poderá atualizar este documento conforme necessário para aprimorar o funcionamento da biblioteca.
